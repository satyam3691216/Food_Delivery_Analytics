# Food Delivery Analytics

## Objective
Analyze restaurant data to find patterns in ratings, cost, and location.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- VS Code

## Dataset

The dataset used in this project is not included in this repository due to GitHub's file size limitations.

You can download the dataset from:
- Kaggle: https://www.kaggle.com/datasets (search: "Zomato Bangalore Dataset")

After downloading:
1. Extract the ZIP file
2. Place the CSV file in the project directory
3. Update the file path in the code if needed
